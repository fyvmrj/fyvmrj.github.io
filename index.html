FAQ


New to FollowYu

Get it

Supported smartphones

FollowYu Messenger is supported on smartphones running Android 4.0 and above.

How to use

Install -> Register -> Message. That's it.


Setup

Register

Start FollowYu app.

Select "Register New Account" option.

Select your country and type your mobile number.

Select a strong password. Password is required to support on SIM free devices.

Tap on Next button to register new account with FollowYu. Your mobile number will be validated with a missed call from FollowYu. Please don't leave app or don't disconnect internet connection while registration is in progress as this may lead to some unexpected behavior.


If you failed to register then check your internet connection, mobile number and try again. Even you can follow troubleshoot faq.


Note: Make sure you will use the validated mobile number and password for future login if you logout or when you use app on other devies.

Login/SIM free devices

Start FollowYu app.

Select your country and type your verified mobile number.

Type your password.

Tap on Next button to continue login.


If you failed to login then check your internet connection, mobile number and try again.


Note: Make sure you used mobile number which is already validated on a SIM device. If not then register Click Here


Contacts

Add/refresh contacts

Tap on "New Conversation".

Tap on "Add Contact".

Click on "Ok" to continue. All your phone numbers will be verified with FollowYu server and added them(which are using FollowYu Messenger) into your contact list.


Note: Make sure you have your friends'(who are using FollowYu app) phone numbers in your phone book. Invite your friends and family to use FollowYu app to have more contacts.

Block a contact

Go to contact details.

Select "Block Contact" from more options.

UnBlock a contact

Go to contact details.

Select "Unblock Contact" from more options.


Conversations

New conversation

Tap on "New Conversation"

Select a contact.

New secure conversation

Tap on "New Conversation"

Select a contact.

Tap on "Secure Conversation" from conversation screen.

Select "OTR". You will be promted to verify contact's finger prints and carefully verify them. All the messages in this conversation will be secured with OTR encryption.

Switch from plain to secure conversation

Go to ongoing conversation.

Tap on "Secure Conversation" from conversation screen.

Select "OTR". You will be promted to verify contact's finger prints and carefully verify them. All the messages in this conversation will be secured with OTR encryption.

Switch from secure to plain conversation

Go to ongoing conversation.

Tap on "Secure Conversation" from conversation screen.

Select "Plain".

Secure all conversations

Go to advance settings from more options

Select "Force end-to-end encryption" option.

In each conversation, now onwards you will be promted to verify contact's finger prints and carefully verify them. All the messages in the conversations will be secured with OTR encryption.


Account

Temporally disable account

Tap on "Account" from more settings.

Long tap on account.

Click on "Temporally disable"

Enable disabled account

Tap on "Account" from more settings.

Long tap on account.

Click on "Enable account"

Change Password

Tap on "Account" from more settings.

Tap on account.

Selelct "Change password" from more options.

Type current and new passwords. Click on Change Password to change password.

Logout

Tap on "Account" from more settings.

Long tap on account.

Click on "Logout" and Ok.

Troubleshoot

1. If something went wrong with internet connection while registration is in progress and landed home screen without welcom message from FollowYu.
2. If you logout from app and not finding login screen back.

Resolution:
Start freshly FollowYu app as just clear app temporary data from phone settings(settings->apps->select FollowYu app->clear temp data). If you clear app temp data then all the messages history will be cleared so carefull.
